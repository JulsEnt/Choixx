Deployment

How to put it online

1

Deploy everything on Render

Create one Docker Web Service. Render serves the website and the Python audio API from the same public URL.

2

No Netlify needed

The frontend files live inside the static folder and are served by FastAPI.

3

Open your Render URL

Visit your .onrender.com link, upload audio, and render directly.

Important legal note

Only upload and transform songs you own, created yourself, licensed, or have clear permission to adapt. Do not use this to redistribute copyrighted music without permission.
